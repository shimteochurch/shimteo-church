# Image Folder — 쉼터 교회

Place your church images in this folder. Each file must match the exact filename listed below.

## Required Images

| File | Purpose | Recommended Size |
|------|---------|-----------------|
| `hero.jpg` | Full-width homepage background | 1920 x 1080 px |
| `pastor.jpg` | Pastor photo for About page | 800 x 800 px (square) |
| `icon.png` | Church icon / logo (also used as favicon) | 512 x 512 px |
| `og-image.jpg` | Social media share preview image | 1200 x 630 px |

## Notes

- **hero.jpg**: If this file is missing, the homepage hero section will show a solid navy background instead.
- **pastor.jpg**: If this file is missing, the About page will show a placeholder shape instead.
- **icon.png**: Used as the browser tab favicon. Replace with your church logo.
- **og-image.jpg**: This image appears when someone shares your website link on Facebook, KakaoTalk, etc.

## Image Tips

- Use JPEG (.jpg) for photos, PNG (.png) for logos with transparency.
- Keep file sizes reasonable (under 500KB each) for fast loading.
- Make sure images are not blurry — use the recommended sizes above as minimum dimensions.
